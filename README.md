# 🤖 AIVORA: BK E-Counseling v1.0

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![PHP](https://img.shields.io/badge/PHP-8.2.12-777BB4?logo=php)
![Laravel](https://img.shields.io/badge/Laravel-12.x-FF2D20?logo=laravel)
![School](https://img.shields.io/badge/SMK-Budi_Bakti_Ciwidey-red)

**BK E-Counseling** adalah platform digital bimbingan konseling cerdas yang dirancang untuk mentransformasi manajemen kedisiplinan dan pembinaan siswa. [cite_start]Aplikasi ini menggantikan pencatatan manual buku besar menjadi sistem digital terpusat yang objektif dan transparan[cite: 306, 362, 365].

[cite_start]Dikembangkan oleh **Tim AIVORA** sebagai syarat kelulusan **Uji Kompetensi Keahlian (UKK) Rekayasa Perangkat Lunak 2025/2026**[cite: 301, 305].

---

## ✨ Fitur Utama (Core Features)

- [cite_start]**Sistem Poin Otomatis**: Pencatatan pelanggaran siswa dengan akumulasi poin otomatis sebagai dasar pemberian sanksi objektif[cite: 366, 532].
- [cite_start]**Booking Konseling Online**: Memudahkan siswa mengajukan jadwal pertemuan bimbingan secara fleksibel[cite: 531, 557].
- [cite_start]**WhatsApp Gateway**: Pengiriman notifikasi otomatis kepada orang tua terkait pelanggaran dan panggilan BK[cite: 533, 559].
- [cite_start]**Cetak Dokumen Formal**: Pembuatan Surat Peringatan (SP1, SP2, SP3) dan Surat Panggilan secara otomatis dalam format PDF/Word[cite: 332, 560].
- [cite_start]**Monitoring Statistik**: Dashboard dengan grafik tren pelanggaran per minggu/bulan untuk evaluasi sekolah[cite: 419, 554].

---

## 💻 Arsitektur Teknologi (Tech Stack)

| Komponen     | Teknologi                                                              |
| :----------- | :--------------------------------------------------------------------- |
| **Backend**  | [cite_start]PHP 8.2.12 & Laravel 12 [cite: 374, 551]                   |
| **Frontend** | [cite_start]Bootstrap 5 & Tailwind CSS [cite: 377, 551]                |
| **Database** | [cite_start]MySQL / MariaDB [cite: 373, 551]                           |
| **Tools**    | [cite_start]VS Code, Git, Composer, XAMPP v3.3.0 [cite: 372, 376, 551] |

---

## ⚙️ Persyaratan Sistem (Hardware)

- [cite_start]**Processor**: Core i5 [cite: 379]
- [cite_start]**RAM**: 8GB [cite: 380]
- [cite_start]**Storage**: Minimal sisa ruang 142GB [cite: 381]

---

## 🛠️ Panduan Instalasi

1. **Persiapan Folder**:
   [cite_start]Simpan folder project di direktori `C:\xampp\htdocs\`[cite: 389].
2. **Konfigurasi Database**:
   [cite_start]Import file `.sql` melalui `phpMyAdmin`[cite: 390, 393].
3. **Konfigurasi Environment**:
   [cite_start]Buka file `.env` dan sesuaikan parameter `DB_DATABASE` dengan nama database Anda[cite: 399, 402].
4. **Menjalankan Aplikasi**:
   Buka terminal dan jalankan:
    ```bash
    php artisan serve
    ```
5. ** Akses di browser melalui http://localhost:8000.**
